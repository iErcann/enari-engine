### Controls
- **W A S D**: Movement
- **1**: First Weapon
- **2**: Second Weapon
- **3**: Knife
- **4**: First Person View
- **5**: Third Person View
- **E**: Respawn (clone)
- **Left Click**: Fire
- **Escape**: Leave mouse lock
- **R**: Reload

 
 
 **I updated the project but it has old Three.js dependencies + Ammo.js has a better alternative now (Rapier.js)**
